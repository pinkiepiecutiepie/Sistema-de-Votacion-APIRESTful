# Solucion Backend para un Sistema de Gestion de Votaciones Estudiantiles
Instrucciones de Ejecución
Descargar el Archivo: Descarga el archivo del proyecto y colócalo en el espacio de trabajo de Eclipse IDE.

Instalar Extensión: Descarga e instala la extensión Spring Tools Suite desde el Marketplace de Eclipse.

Configurar Base de Datos: Ejecuta el servidor SQL utilizando XAMPP u otro conector SQL de tu preferencia.

Acceder a la Consola de tu Base de Datos: Si usas XAMPP, abre tu navegador y ve a localhost/phpmyadmin/.

Crear Base de Datos: En la consola de SQL, crea una base de datos llamada 'voto_electronico_db'.

Abrir el Proyecto en Eclipse: Abre el archivo del proyecto descargado en Eclipse.

Ejecutar la Aplicación: Haz clic derecho sobre la clase principal del proyecto y selecciona Run As > Spring Boot App.

Verificar Creación de Tablas: Regresa a la consola de SQL y verifica que las tablas se hayan creado correctamente; deberían aparecer en la base de datos llamada 'voto_electronico_db'.
